## Mo2017-p234-N12
**题：** 仔细观察题图，问号的地方应该填入（  ）。

|第一列| |第二列| |第三列| |第四列|
|:-:|:-:|:-:|:-:|:-:|:-:|:-:|
|7| |3| |7| |?|
|6| |9| |6| |?|
|8| |8| |8| |?|
|2| |6| |9| | |
|9| |7| | | | |
|3| | | | | | |

**解：**
仔细观察各列，可知：剔除前列中的最小值，并逆序重排在后一列。

故知第四列为9，8，7。

**答：**最后一列为{9，8，7}。
